# Send-Ready Note

This is the integrated public reproducibility package for the static flagship in
Finite Consequence Formation.

It is intended to be directly usable for:

- GitHub repository upload
- Zenodo dataset upload
- direct sharing with reviewers or faculty

Included layers:

1. `core_repro/`
   - executable reproduction scripts
2. `data_release/`
   - frozen public node/edge/contact/Q tables
3. `audit_archive/`
   - frozen flagship and certificate summaries
4. `results/`
   - frozen public headline result tables
5. `expected_outputs/`
   - machine-checkable PASS/FAIL targets
6. `source_evidence/`
   - extracted certificate-source scripts and source summaries

Boundary:

- static hidden-twin load-repair only
- not dynamic action CREATE

Recommended public repo name:

`FCF-static-flagship-repro`
