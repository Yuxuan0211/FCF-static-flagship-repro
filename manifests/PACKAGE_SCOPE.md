# Package Scope

This integrated package is built from:

1. `FCF_static_flagship_repro_public_v2`
   - primary public repro package with frozen public data tables
2. extracted public certificate-source artifacts
   - added under `source_evidence/`

Design choice:

- `public_v2` remains the scientific base
- integration only adds verification convenience, source-evidence context,
  and repo/send readiness

No scientific result values were altered during integration.
